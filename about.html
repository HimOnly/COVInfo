About this APP

This web app project is built or created by God'spower Alo as an Open Source app. This SERVICE is made available at no cost and is intended for use as it is!


This web app project is to help serve the world latest news, up to date information and live statistics about the COVID-19 (Corona Virus) Pandemic that is ravaging the world.


This project was actually born at the #BuildforCOVID19 Global Online Hackathon organised by Facebook. It was a Global Hackathon for developers across the globe to build software solutions that drive social impact, with the aim of tackling some of the challenges related to the current coronavirus (COVID-19) pandemic.
